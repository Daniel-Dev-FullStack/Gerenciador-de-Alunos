# Gerenciador de Alunos
Gerenciador de Alunos desenvolvido com HTML, CSS e JavaScript puro, focado no uso prático de arrays, objetos e métodos como map, filter, reduce e forEach. O sistema permite cadastrar alunos, calcular médias, aplicar filtros por situação e atividade, e exibir uma análise geral da turma de forma dinâmica, sem frameworks.
